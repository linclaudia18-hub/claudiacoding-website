# 🌐 Personal Landing Page - claudiacoding.com

Mi página web personal con estética de terminal interactiva y animaciones en CSS/JS.

### 🛠️ Tecnologías utilizadas:
* **HTML5** - Estructura semántica.
* **CSS3** - Animaciones de burbujas flotantes, efecto Neumorphism/Dark Mode y diseño Responsive.
* **JavaScript (Vanilla)** - Lógica de terminal y dinámicas de texto.

🔒 *No cookies, no tracking.*
